# AI Resume Analyzer & Job Matcher

## 🚀 Features
- Upload Resume (PDF)
- Compare with Job Description
- Match Score
- Missing Skills

## 🛠️ Tech Stack
- Python
- Streamlit
- spaCy
- scikit-learn

## ▶️ How to Run

pip install -r requirements.txt
streamlit run app.py
